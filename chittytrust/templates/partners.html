API Reference

RESTful API for seamless ChittyID integration

Core Endpoints

GET /api/trust/{user_id}
Get comprehensive trust score for a user
POST /api/verification/request
Create new verification request
GET /api/trust/history/{user_id}
Get historical trust data with trends
POST /api/events/record
Record trust-affecting events

Marketplace Endpoints

GET /api/marketplace/requests
List available verification requests
POST /api/marketplace/claim/{id}
Claim verification request
PUT /api/verification/complete/{id}
Complete verification with results
GET /api/verifiers/profiles
Get available verifier profiles

Integration Examples

Quick-start code snippets for popular frameworks

JavaScript SDK 
// Install: npm install @chitty/trust-sdk
import { ChittyTrust } from '@chitty/trust-sdk';

const chitty = new ChittyTrust({
  apiKey: 'your-api-key',
  environment: 'production'
});

// Get user trust score
const trustScore = await chitty.getTrustScore('user123');
console.log(trustScore.composite); // 78.5

// Create verification request
const request = await chitty.createVerification({
  type: 'identity',
  title: 'Verify government ID',
  reward: 25.00
});
Python SDK 
# Install: pip install chitty-trust
from chitty_trust import ChittyClient

client = ChittyClient(
    api_key="your-api-key",
    environment="production"
)

# Get trust score with history
trust_data = client.get_trust_score(
    user_id="user123",
    include_history=True,
    days_back=30
)

print(f"Trust Score: {trust_data.composite}")
print(f"Trend: {trust_data.trends.change}%")
React Component 
import { ChittyTrustWidget } from '@chitty/react-components';

function UserProfile({ userId }) {
  return (
    <div>
      <ChittyTrustWidget
        userId={userId}
        showHistory={true}
        theme="dark"
        onScoreUpdate={(score) => {
          console.log('Trust score updated:', score);
        }}
      />
    </div>
  );
}
cURL Example 
# Get trust score
curl -X GET \
  "https://api.chittyid.com/v1/trust/user123" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json"

# Create verification request
curl -X POST \
  "https://api.chittyid.com/v1/verification/request" \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "identity",
    "title": "Verify government ID",
    "reward": 25.00
  }'

Real-time Updates

Stay synchronized with webhook notifications

Webhook Events
  • trust.score_updated - Trust score changed
  • verification.completed - Verification finished
  • verification.claimed - Request claimed by verifier
  • user.level_changed - Trust level upgraded/downgraded

  "event": "trust.score_updated",
